# User Stories

## Template

## User Stories List

### 1
**As a** admin  
**I need** to create a customer account  
**So that** I can store customer data  

### 2
**As a** user  
**I need** to view customer data  
**So that** I can see customer information  

### 3
**As a** admin  
**I need** to update customer data  
**So that** I can keep information accurate  

### 4
**As a** admin  
**I need** to delete customer data  
**So that** I can remove inactive customers
