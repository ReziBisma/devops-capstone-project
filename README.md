# Customer Account Service

Microservice untuk mengelola akun pelanggan menggunakan REST API, Docker, dan Kubernetes.

## Features
- Create customer
- Read customer data
- Update customer
- Delete customer

## Tech Stack
- Python (Flask)
- Docker
- Kubernetes
- GitHub Actions (CI/CD)
